# WLauncher (WotLK Launcher)

Launcher de escritorio para el cliente de World of Warcraft: Wrath of the Lich King (3.3.5a) servidor privado. Creado con Electron, React, TypeScript, Vite y Tailwind CSS.

## Qué hace:

- Valida y guarda la ruta local del cliente del juego WoW.
- Reescribe `realmlist.wtf` con el reino (servidor) seleccionado por el jugador.
- Limpia la carpeta `Cache` y crea copias de seguridad comprimidas en `.zip` de la carpeta `WTF`.
- Ejecuta `Wow.exe` directamente como un proceso independiente.
